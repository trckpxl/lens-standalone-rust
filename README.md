# Lens Standalone

A lightweight, high-performance Rust library and CLI to use Google Lens OCR without a headless browser.

## Installation

```bash
cargo add lens-standalone

# Or install the CLI tool globally
cargo install lens-standalone
```

## Usage

```bash
lens-standalone image.png
```
